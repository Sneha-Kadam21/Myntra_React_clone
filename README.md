# Myntra Clone (ReactJS)

This is a front-end clone of the Myntra e-commerce platform built using ReactJS. The project simulates product listings, cart management, and UI components using modern front-end development practices.

## 🚀 Project Overview

This project replicates the core structure and UI of Myntra using **ReactJS**. It includes reusable components, routing, and state management using React Hooks.

## 🔧 Tech Stack

- ReactJS (Functional Components)
- JavaScript (ES6+)
- CSS3
- React Router (if used)
- [Optional] Context API / Redux (if used)

## 📌 Features

- Modular UI with reusable components
- Product listing cards
- Add to Bag simulation using React state
- Basic routing/navigation 
- Responsive layout
